DIR

Tipo: CMD

Descrição: Lista os arquivos e subdiretórios de um diretório.

Sintaxe: DIR [unidade:][caminho][arquivo] [/A[[:]atributos]] [/B] [/C] [/D] [/L] [/N] [/O[[:]ordem]] [/P] [/Q] [/R] [/S] [/T[[:]campo]] [/W] [/X] [/4]

dir C:\Users\SeuUsuario\Documents
dir /b /s *.txt

Observações: O parâmetro `/B` exibe os nomes sem cabeçalho ou resumo. O `/S` lista os arquivos em subdiretórios.

COPY

Tipo: CMD

Descrição: Copia um ou mais arquivos para outro local.

Sintaxe: COPY [/D] [/V] [/N] [/Y | /-Y] [/Z] [/L] [/A | /B ] origem [/A | /B] [+ origem [/A | /B] [+ ...]] [destino [/A | /B]]

copy "C:\caminho\arquivo.txt" "D:\backup\"

Observações: Use `/Y` para suprimir o prompt de confirmação de sobrescrita.

MOVE

Tipo: CMD

Descrição: Move um ou mais arquivos de um diretório para outro. Também pode renomear arquivos.

Sintaxe: MOVE [/Y | /-Y] [unidade:][caminho]nomearquivo1[, ...] destino

move "C:\temp\relatorio.docx" "C:\Documentos\Relatorios\"

Observações: Para renomear, use `move nome_antigo.txt nome_novo.txt`.

DEL / ERASE

Tipo: CMD

Descrição: Exclui um ou mais arquivos.

Sintaxe: DEL [/P] [/F] [/S] [/Q] [/A[[:]atributos]] nomes

del /S /Q C:\Temp\*.log

Observações: `/S` exclui de subdiretórios. `/Q` não pede confirmação. Use com extrema cautela.

MKDIR / MD

Tipo: CMD

Descrição: Cria um novo diretório.

Sintaxe: MKDIR [unidade:]caminho ou MD [unidade:]caminho

mkdir "C:\Novos Projetos\Projeto Alpha"

Observações: Cria diretórios pai se não existirem.

RMDIR / RD

Tipo: CMD

Descrição: Exclui um diretório.

Sintaxe: RMDIR [/S] [/Q] [unidade:]caminho ou RD [/S] [/Q] [unidade:]caminho

rmdir /S /Q "C:\Temp\PastaAntiga"

Observações: `/S` remove a árvore de diretórios inteira (todos os arquivos e subpastas). `/Q` modo silencioso.

TYPE

Tipo: CMD

Descrição: Exibe o conteúdo de um arquivo de texto na tela.

Sintaxe: TYPE [unidade:][caminho]nomedoarquivo

type C:\logs\app.log

Observações: Ideal para visualizar arquivos de configuração ou log rapidamente.

PING

Tipo: CMD

Descrição: Verifica a conectividade de rede com outro dispositivo (IP ou domínio).

Sintaxe: PING [-t] [-a] [-n contagem] [-l tamanho] [-f] [-i TTL] [-v TOS] [-r contagem] [-s contagem] [[-j listaHost] | [-k listaHost]] [-w tempoLimite] nomeDestino

ping google.com
ping -t 192.168.1.1

Observações: `-t` faz o ping contínuo até ser interrompido (Ctrl+C).

IPCONFIG

Tipo: CMD

Descrição: Exibe a configuração do protocolo IP (TCP/IP) dos adaptadores de rede.

Sintaxe: IPCONFIG [/allcompartments] [/all] [/renew [adaptador]] [/release [adaptador]] [/renew6 [adaptador]] [/release6 [adaptador]] [/flushdns] [/displaydns] [/registerdns] [/showclassid adaptador] [/setclassid adaptador [IDClasse]]

ipconfig /all
ipconfig /flushdns

Observações: `/all` mostra informações detalhadas. `/flushdns` limpa o cache de resolução de DNS.

TRACERT

Tipo: CMD

Descrição: Determina a rota até um destino, mostrando os saltos (routers) intermediários.

Sintaxe: TRACERT [-d] [-h saltos_máximos] [-j listaHost] [-w tempoLimite] nomeDestino

tracert -d microsoft.com

Observações: `-d` impede a resolução de nomes para endereços IP, acelerando o processo.

NETSTAT

Tipo: CMD

Descrição: Exibe estatísticas de protocolo e conexões de rede ativas.

Sintaxe: NETSTAT [-a] [-b] [-e] [-f] [-n] [-o] [-p proto] [-r] [-s] [-x] [-t] [intervalo]

netstat -ano

Observações: `-a` mostra todas as conexões e portas de escuta. `-n` mostra endereços e números de porta numericamente. `-o` mostra o ID do processo (PID) para cada conexão.

SYSTEMINFO

Tipo: CMD

Descrição: Exibe informações detalhadas sobre a configuração do hardware e do software do computador.

Sintaxe: SYSTEMINFO [/S sistema [/U usuário [/P [senha]]]] [/FO formato] [/NH]

systeminfo

Observações: Útil para auditoria e diagnóstico. Pode levar alguns segundos para coletar todas as informações.

TASKLIST

Tipo: CMD

Descrição: Exibe uma lista de todos os processos em execução no momento no sistema local.

Sintaxe: TASKLIST [/S sistema [/U usuário [/P [senha]]]] [/M [módulo] | /SVC | /V] [/FI filtro] [/FO formato] [/NH]

tasklist /svc

Observações: `/svc` exibe os serviços hospedados em cada processo. `/fi "PID eq 1234"` filtra por um PID específico.

TASKKILL

Tipo: CMD

Descrição: Encerra uma ou mais tarefas ou processos.

Sintaxe: TASKKILL [/S sistema [/U usuário [/P [senha]]]] { [/FI filtro] [/PID processid | /IM imagename] } [/T] [/F]

taskkill /F /IM notepad.exe
taskkill /F /PID 5844

Observações: `/F` força o término do processo. Use com cuidado, pois pode causar perda de dados.

CHKDSK

Tipo: CMD

Descrição: Verifica um disco em busca de erros e exibe um relatório de status.

Sintaxe: CHKDSK [volume[[caminho]nomedoarquivo]] [/F] [/V] [/R] [/X] [/I] [/C] [/L[:tamanho]] [/B
